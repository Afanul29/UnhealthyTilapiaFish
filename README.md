# Artificial Intelligence-Based Health Classification of Nile Tilapia Using Digital Image Processing and Deep Learning

## Overview

This research presents an Artificial Intelligence (AI)-based system for
automatically classifying the health condition of Nile Tilapia
(*Oreochromis niloticus*) using digital image processing and deep
learning techniques. The study aims to assist fish farmers in
identifying healthy and diseased fish through image analysis, reducing
reliance on manual inspection and improving disease monitoring
efficiency.

## Research Title

**Artificial Intelligence-Based Health Classification of Nile Tilapia
(*Oreochromis niloticus*) Using Digital Image Processing and Deep
Learning: A Case Study in Blawi Village, Lamongan Regency, Indonesia**

## Background

Nile Tilapia is one of the most widely cultivated freshwater fish
species in Indonesia. Fish health plays a critical role in aquaculture
productivity and economic sustainability. Traditionally, disease
detection is performed manually based on visual observation, which can
be time-consuming, subjective, and dependent on expert knowledge.

Recent advances in Artificial Intelligence, Computer Vision, and Deep
Learning enable automated image-based disease detection systems that can
classify fish health conditions quickly and accurately.

## Objectives

1.  Develop an image-based health classification system for Nile
    Tilapia.
2.  Apply digital image processing techniques for fish image analysis.
3.  Utilize deep learning algorithms for automatic disease
    classification.
4.  Evaluate model performance using standard metrics.
5.  Support sustainable aquaculture practices.

## Dataset

Dataset: Enhancing Disease Detection in Nile Tilapia (Kaggle)

Disease Categories: - Healthy - Columnaris - Parasite - Septicemia -
Streptococcosis - Tilapia Lake Virus (TiLV)

## Methodology

### Data Acquisition

Fish images are collected from a public dataset and may be supplemented
with local images from Blawi Village.

### Image Preprocessing

-   Resize images to 224×224 pixels
-   Normalize pixel values
-   Data augmentation (rotation, zoom, horizontal flip)

### Dataset Split

-   Training Set: 80%
-   Validation Set: 20%

### Deep Learning Model

MobileNetV2 Transfer Learning is used to improve accuracy while reducing
training time.

### Evaluation Metrics

-   Accuracy
-   Precision
-   Recall
-   F1-Score
-   Confusion Matrix

## Workflow

Image Acquisition → Image Preprocessing → Data Augmentation → Dataset
Splitting → Transfer Learning (MobileNetV2) → Model Training → Model
Evaluation → Fish Health Classification
